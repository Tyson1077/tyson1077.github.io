git创建仓库

创建与git账户同名公有仓库

repository仓库名称：.github.io

git绑定github账户

//git config命令的--global参数表示你这台机器上所有的Git仓库都会使用这个配置，当然也可以对某个仓库指定不同的用户名和Email地址

生成ssh公钥绑定github_sshkey

ssh-keygen -t rsa -C "github邮箱"
//由于github要求推送同步需要验证是否为合法用户，为省时间采用配对公私钥验证账户密码保证推送同步的合法性

通过密钥登录连接github

ssh git@github.com
gitbash连接后出现以下表示git与github账户连接成功

nodejs安装

//hexo基于nodejs，安装后自动添加环境变量和npm
检验nodejs和npm是否安装
node -v
npm -v

hexo安装

创建文件夹作为hexo根目录绝对路径

进入hexo根目录操作
cd hexo/

通过windows powershell安装hexo：shift+右键

npm install -g hexo-cli
成功

出现报错：

  • win10默认策略下无法执行powershell脚本，需要更改设置允许本地powershell在不签名情况下运行 参考https://blog.csdn.net/a12355556/article/details/115856367

成功

初始化hexo

//本地hexo预览地址 localhost:4000
//新建一个网站。如果没有指定文件夹，hexo默认在目前的文件夹建立网站。
hexo init
hexo -v
新建页面//new
hexo n xxx
生成静态页//generate
hexo g
启动本地预览//server
hexo s
关闭本地服务器
ctrl+c
列出网站结构信息
hexo list xxx

配置hexo网页

安装hexo git模块拓展
npm i hexo-deployer-git
关联github：站点_config.yml

部署hexo上线

安装git部署插件 同步本地部署到仓库
npm install hexo-deployer-git --save
更新hexo缓存
hexo clean
生成静态页//generate
hexo g
部署操作//deploy
hexo d

主题更新

cd themes/next
git pull

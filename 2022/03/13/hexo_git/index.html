git创建仓库

创建公有仓库与git账户同名

repository仓库名称：.github.io

git绑定github账户

//git config命令的--global参数表示你这台机器上所有的Git仓库都会使用这个配置，当然也可以对某个仓库指定不同的用户名和Email地址

生成ssh公钥绑定github_sshkey

ssh-keygen -t rsa -C "github邮箱"
//由于github要求推送同步需要验证是否为合法用户，为省时间采用配对公私钥验证账户密码保证推送同步的合法性

通过密钥登录连接github

ssh git@github.com
gitbash连接后出现以下表示git与github账户连接

nodejs安装

//hexo基于nodejs，安装后自动添加环境变量和npm
检验nodejs和npm是否安装
node -v
npm -v

hexo安装

创建文件夹作为hexo根目录绝对路径

通过windows powershell安装hexo：shift+右键

npm install -g hexo-cli
成功

出现报错：

  • win10默认策略下无法执行powershell脚本，需要更改设置允许本地powershell在不签名情况下运行 参考https://blog.csdn.net/a12355556/article/details/115856367

成功

初始化hexo

//本地hexo地址 localhost:4000
hexo init
hexo -v
新建页面//new
hexo n xxx
生成静态页//generate
hexo g
启动预览//server
hexo s
关闭本地服务器
ctrl+c

配置hexo网页

安装hexo git模块拓展
npm i hexo-deployer-git
关联github：站点_config.yml

部署hexo上线

更新hexo根目录缓存
hexo clean
生成静态页//generate
hexo g
部署操作//deploy
hexo d

hexo自定义

更换主题

克隆主题到hexo根目录中
cd hexo/hexo
git clone https://github.com/theme-next/hexo-theme-next themes/next
修改站点_config.yml主题名
theme:主题名
修改主题_config.yml样式
Scheme Settings

主题更新

cd themes/next
git pull

修改基本信息配置：站点_config.yml

基本变量

基本概念：

  1. 变量在内存中存储数据
  2. 变量存在名字
  3. 变量通过名字访问数据
  4. 变量可改变：数据

使用变量：

  1. 定义变量：不需要任何关键字定义变量：即php不用声明数据类型$
  2. 访问输出变量：通过变量名在内存中找到存储值后输出echo
  3. 修改变量
  4. 删除变量unset($)

变量标识符规则：

预定义变量

//即关键字，提前定义的变量，存储系统中频繁使用的数据。都是数组

原理：

  1. 寻址变量$a，解析变量$a得结果：b
  2. 变量标识符$ 绑定结果b：$b
  3. 解析变量$b

传值变量

传统赋值：值传递$

//原变量赋值保存，新变量赋新值保存。两个变量指向两个数据段内存址，占两个数据段；两个变量无关系

原理：

引用赋值：引用传递&$

//原变量赋值内存地址传递给新变量。两个变量指向同一个内存附位址；两个变量是动态同值，同一个数据段内存址

原理：

内存址分区

  • 代码段：存储代码的内存区域 //不执行
  • 数据段：存储(预)分配的普通数据代码内存地址 //可执行 分全局区静态区
  • 栈区stack：内存操作运行程序区域。范围小m速度快 //仅执行
  • 堆区heap：内存中操作管理动态分配回收内存区域。存储复杂数据，范围大速度慢

数据类型

php是一种弱类型语言，数据类型特指(赋值)存储的数据类型
//1字节B=2节b=8位bit;

基本类型

int整型：表示整数，包含+-表示八O十D十六进制H数，内存中4个字节存储 //共32位二进制数最多存储42亿个正负数值
float浮点型：表示小数，超过字符的整型，8个字节存储
string字符串型：‘’与””中表示，”“可解析变量
bool布尔型：输出true与false，进行布尔运算输出true为1 false为0

复合类型

object对象型：存放面向对象
array数组型：存放多个数据合成为组

特殊型

resource资源型：索引php外部资源
NULL空型

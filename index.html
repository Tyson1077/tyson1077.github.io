数据库

按数据结构管理存储计算机单元仓库；

数据库分类

网络数据库

借助计算机网络技术存储数据；

层级数据库

无回路单定向有序树层级数据图；

关系数据库：主要

基于集合线性代数建立于关系模型中处理数据关系；

区分于存储介质

非关系数据库：内存存储

关系型数据库：磁盘存储

关系数据结构：以二维表形式存储数据；
|姓名|年龄|
|—|—|
|B1|B2|

关系操作集合：指某方式操作关联管理的存储数据，如增删改查(sql指令集)；
获取年龄：通过已知条件；
select 年龄 from 二维表 where 已知条件表 = 已知条件；
关系完整性约束：数据内部实体，数据间实体一一关联对应；
表内约束：对应行列存放对应类型数据(内部严谨联系)；
表间约束：关系表各式体存在对应关联(外键外联);

常见关系数据库

小型：微软access/sqlite
中型：微软sqlserver/mysql
大型：oracle/db2

SQL

structured query language结构化查询语言，皆为编程关系表数据管理检索而设计；

sql语言分类

数据查询语言data query language:DQL

通过dql保留字与sql操作语句：确定获取表数据，如何输出给程序；
常见保留字：where/order by/group by/having
保留指令集：select/show

数据操作语言data manipulation language:DML

增删改数据中行；
指令集：insert/delete/update

事务处理语言transaction:TPL

确保dml操作表所有数据的更新得到安全记录；
指令集：BEGIN TRANSACTION/COMMIT/ROLLBACK

数据控制语言DCL

通过grant授权分配/revoke回收授权确定用户与用户组访问数据库对象的权限管理控制；
指令集：grant/revoke

数据定义语言DDL

通过数据结构表管理进行索引alter改变：新建create table数据表/删除drop table数据表
指令集：create table/drop table
